CAD Lesson 2 – Design Intent & Parametric Modeling

Design intent is about building models that update correctly when dimensions change. In Onshape, this means using constraints and dimensions that reflect how the part should behave, not just how it looks once.

What is Design Intent?

  • Thinking about how a part might change in the future.
  • Using symmetry, equal constraints, and patterns instead of manual duplication.
  • Placing dimensions on critical features, not random edges.

Parametric Modeling

Parametric models are driven by parameters (dimensions, relations, variables). Changing a dimension should update the model predictably.

  • Use Equal to keep repeated features the same size.
  • Use Symmetric to keep geometry centered.
  • Use Patterns instead of copying features manually.

Best Practices

  • Dimension to centerlines and reference geometry when possible.
  • Avoid over‑defining sketches with redundant dimensions.
  • Think: “If this dimension changes, does the model still make sense?”
